# @zephrui/avatars

[![npm](https://img.shields.io/npm/v/@zephrui/avatars?color=blue)](https://www.npmjs.com/package/@zephrui/avatars)
[![license](https://img.shields.io/npm/l/@zephrui/avatars)](https://github.com/akhilllkrishnan/zephr/blob/main/LICENSE)

Deterministic avatar generation for the Zephr design system. Included as a dependency of `@zephrui/ui-react`.

## Install

```sh
npm install @zephrui/avatars
```

Most users get this automatically via `@zephrui/ui-react`.
