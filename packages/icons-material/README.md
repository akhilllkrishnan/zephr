# @zephrui/icons-material

[![npm](https://img.shields.io/npm/v/@zephrui/icons-material?color=blue)](https://www.npmjs.com/package/@zephrui/icons-material)
[![license](https://img.shields.io/npm/l/@zephrui/icons-material)](https://github.com/akhilllkrishnan/zephr/blob/main/LICENSE)

Material-style icon set for the Zephr design system. Included as a dependency of `@zephrui/ui-react`.

## Install

```sh
npm install @zephrui/icons-material
```

Most users get this automatically via `@zephrui/ui-react`.
