# @zephrui/logos

[![npm](https://img.shields.io/npm/v/@zephrui/logos?color=blue)](https://www.npmjs.com/package/@zephrui/logos)
[![license](https://img.shields.io/npm/l/@zephrui/logos)](https://github.com/akhilllkrishnan/zephr/blob/main/LICENSE)

Logo provider and fallback system for the Zephr design system. Included as a dependency of `@zephrui/ui-react`.

## Install

```sh
npm install @zephrui/logos
```

Most users get this automatically via `@zephrui/ui-react`.
